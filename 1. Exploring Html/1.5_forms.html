Forms

Forms are used when we want to collect information from users(visitors) on our website. This includes examples such as an pplication form, sign up, shopping online etc.
There are several types of form controls used to collect data to your site.

  1. Adding Text
    • Text Input(single line)
    • Password Input(single line but masks characters)
    • Text Area(used for comments or paragraphs)
  2. Making choice
    • Radio Buttons(circle options allowing user to select only one)
    • CheckBoxes(squares allowing user to select multiple options)
    • Drop-down boxes(list allowing user to choose one given option)
  3. Submitting Forms
    • Submit Buttons
    • Image Buttons(same as ubmit button, except that an image is used)
  4. Uploading Files
    • File upload(images, pdt, etc)

Form Structure

form tag
Open tag for every form, must carry the action, method and id attributes.
Action attribute points to URL of the page on the server that will receive the information when the form is submitted.
Method attribute takes the get or post value.
The get method takes value in the form and appends them at the end of the URL in the action attribute. It is ideal for short forms(search boxes) and retrieving ata from the web server(not adding or deleting data)
The post method takes value in the form and sends them as HTTP headers, used when uploading a file, when having a long form, passwords(sensitive data), or adding or deleting information from a database
id attribute is used to identify this form from other elements
