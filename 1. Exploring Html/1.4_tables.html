Building Tables

A table is made up of grids( blocks). In order to build a table we need the to write out the table row by row using

  • table tag
  • th tag (table heading)
  • tr tag (table row)
  • td tag (table data)

Example of a basic table(borders are not showing as css is not yet applied)

Chicken Computer Mouse
Human Pencil Microphone
Laptop House Pizza
Keyboard

Example of a table with a headings

th tag is used when we give headings to rows and columns. It usually has the scope attribute is also used to indicate whether the heading is for a row or col. Do not mistaken th with thead

Lenovo i5 R15,000 R10,000
Lenovo i3 R9,800 R7,200
