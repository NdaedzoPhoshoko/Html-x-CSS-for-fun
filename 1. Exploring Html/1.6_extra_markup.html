Meta tag

Meta tag is an empty element used to describe information about a page, it is included on the head element

it has attributes such as:

  • name and content
  • http-equiv and content
  • charset
examples include:
name="description" content="put description here"
name="keywords" content="keyword1, keyword2" used to help search engines identify webpage
name="robots" content="noindex" used to not allow search engines to identify web page
name="robots" content="nofollow" used to allow search engines to identify web page, not related links
http-equiv="author" content="Ndaedzo Clement Phoshoko"
http-equiv="pragma" content="no-cache" sed to not allow caching the web page
http-equiv="expires" content="Sun, 31, Dec 2024 23:59:59 GMT" used to specify date and time for captured captured to ellapse

Html5 Video tag

This element came with html5, not all browsers support it. It is advisable to use it and flash altogether to favor those with visual and/or physical impairments. It uses these attributes and values

  • src - specifies the path to the video
  • poster - an image shown before the video plays
  • width & height
  • controls - allows browser to offer its own controls
  • autoplay - video starts to play automatically
  • preload - what to do when browser loads, values can be
    1. none - not load until user presses play
    2. auto - browser downloads the video when page loads( not recommended as this reduces speed of a website, nuless it is used on a streaming site).
    3. metadata - browser shows information about the video( this includes size, duration, track list, etc).
